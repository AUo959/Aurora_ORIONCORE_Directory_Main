## CONFIDENCE CALIBRATION GUIDE

### Numerical Bands
| Range | Label | Criteria |
|-------|-------|----------|
| 0.90-1.00 | High | Multi-source corroboration + historical precedent |
| 0.70-0.89 | Mod-High | Dual-source verification + strong theory |
| 0.50-0.69 | Moderate | Single reliable source OR multiple uncertain |
| 0.30-0.49 | Low-Mod | Limited evidence, high uncertainty |
| 0.00-0.29 | Low | Speculation, early warning, uncorroborated |

### Component Scores
```
Composite Confidence = (DQ × SR × MR × TS)^(1/4) × Coherence

DQ = Data Quality (source timeliness, corroboration)
SR = Source Reliability (historical accuracy)
MR = Methodological Rigor (SAT application, bias mitigation)
TS = Temporal Stability (forecast consistency over time)
Coherence = Quantum coherence score (scenario concentration)
```

### Quantum Coherence Formula
```
Coherence = 1 - (Σᵢ pᵢ × log₂(1/pᵢ)) / log₂(N)

High coherence (>0.8) = Single scenario dominates
Low coherence (<0.4) = Multiple plausible futures
```

## COMMON BIAS TRAPS

### Cognitive Biases to Monitor
1. **Anchoring:** First assessment sticks too hard
   - *Mitigation:* Fresh eyes review, explicit prior challenge

2. **Confirmation bias:** Seek evidence supporting initial view
   - *Mitigation:* ACH forces considering contradictory evidence

3. **Availability heuristic:** Recent events over-weighted
   - *Mitigation:* Historical base rates, statistical anchoring

4. **Groupthink:** Consensus silences dissent
   - *Mitigation:* Devil's advocacy, Team A/B, anonymous polling

5. **Mirror imaging:** Assume adversary thinks like us
   - *Mitigation:* Red Team, cultural expertise consultation

6. **Recency bias:** Latest intelligence over-valued
   - *Mitigation:* Temporal weighting, trend analysis

### Debiasing Protocol
- Blind review of assessments (remove analyst names)
- Adversarial collaboration (pair optimist + pessimist)
- Track record scoring (accountability for past forecasts)
- Brier score monitoring (calibration feedback)

## MATHEMATICAL QUICK REFERENCE

### Bayes' Theorem (Probability Updates)
```
P(H|E) = P(E|H) × P(H) / P(E)

H = Hypothesis ("invasion in 30 days")
E = Evidence (new intelligence)
P(H) = Prior probability
P(E|H) = Likelihood of evidence given hypothesis
P(H|E) = Posterior probability (updated belief)
```

### Brier Score (Calibration Metric)
```
BS = (1/N) × Σᵢ (fᵢ - oᵢ)²

fᵢ = Forecast probability (0.0-1.0)
oᵢ = Outcome (0 or 1)
N = Number of forecasts

Perfect calibration: BS = 0.0
Random guessing: BS ≈ 0.25
QGIA target: BS < 0.10
```

### Confidence Interval Calculation
```
95% CI = μ ± 1.96σ

μ = Mean probability estimate
σ = Standard deviation across scenarios
1.96 = Z-score for 95% confidence

Example: P = 0.65, σ = 0.08
95% CI: [0.49, 0.81]
```

## COMMON SCENARIO ARCHETYPES

### 1. Salami Slicing (23.4% of conflicts)
**Pattern:** Incremental territorial gains below response threshold  
**Examples:** Crimea 2014, South China Sea islands  
**Warning signs:** Gradual military buildup, diplomatic normalization attempts  
**Countermeasures:** Clear red lines, cumulative response doctrine

### 2. Fait Accompli (18.7%)
**Pattern:** Rapid seizure, present international community with done deal  
**Examples:** Prague 1968, Kuwait 1990  
**Warning signs:** Surprise military exercises, communication blackouts  
**Countermeasures:** Pre-positioned forces, tripwire deployments

### 3. Proxy War Escalation (14.2%)
**Pattern:** Indirect confrontation via client states/non-state actors  
**Examples:** Syria, Yemen  
**Warning signs:** Arms transfers, advisory deployments, financial flows  
**Countermeasures:** Counter-proxy support, escalation management

### 4. Deterrence Failure Cascade (11.8%)
**Pattern:** Bluff called, credibility collapse triggers wider conflict  
**Examples:** World War I, Korean War  
**Warning signs:** Repeated challenges, resolve questioning, commitment traps  
**Countermeasures:** Costly signaling, limited demonstrations of force

### 5. Alliance Entrapment (9.6%)
**Pattern:** Dragged into conflict by alliance obligations  
**Examples:** World War I (extended), Georgia 2008 (near-miss)  
**Warning signs:** Ally risk-taking, expectation of support, ambiguous commitments  
**Countermeasures:** Clarity of obligations, private restraint of allies

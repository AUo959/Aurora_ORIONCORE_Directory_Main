# Skill Boundaries

## ZIPWIZ Governor handles

- ZIPWIZ packaging/export governance artifacts
- ZIPWIZ beacon capsule checks
- ZIPWIZ runtime alignment warnings (ZIPWIZ-only)
- ZIPWIZ evolution timeline synthesis

## Delegate to other skills

- `aurora-script-governor`
  - Use for broad script hazard remediation, setup hardening, and script canonicalization.

- `aurora-repo-stabilizer`
  - Use for repo-ops stabilization: hooks, CI reliability, trust matrix, maintenance pipelines.

- `threadcore-governor`
  - Use for THREADCORE checkpoint/continuity/payload/delta/beacon artifact governance.

## Guardrail

ZIPWIZ Governor may identify runtime alignment drift in ZIPWIZ modules, but must not expand into repo-wide script remediation workflows.
